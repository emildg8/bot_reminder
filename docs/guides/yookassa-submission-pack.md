# ЮKassa: пакет для отправки анкеты

Готовый набор текстов и материалов для страницы «Платежи в телеграм-боте».

**Оплата в боте:** Telegram Stars через `/thanks` (не виджет ЮKassa). См. [stars-tips.md](stars-tips.md).

## 1) Тексты для формы (копипаст)

### Что вы продаёте

`Цифровой сервис: Telegram-бот для напоминаний`

### Ссылка на бота / юзернейм

`@break_remind_bot`

### Краткое описание сервиса

`Reminder — Telegram-бот для личных и групповых напоминаний. Пользователь может отправить добровольную поддержку проекта (донат) через встроенную оплату Telegram Stars. После оплаты пользователь сразу получает подтверждение в чате. Доставка физического товара отсутствует, услуга полностью цифровая.`

### Что получает пользователь после оплаты

`Пользователь получает немедленное подтверждение оплаты в чате бота (сообщение благодарности). Платёж является добровольной поддержкой проекта и не требует отдельной доставки или ожидания.`

### Модель расчётов

`Разовый платёж (донат) с фиксированными суммами и произвольной суммой в допустимом диапазоне (Telegram Stars).`

## 2) Подписи к скриншотам

- `Скрин 1 — Экран бота с кнопкой поддержки (/thanks)`
- `Скрин 2 — Выбор суммы поддержки`
- `Скрин 3 — Подтверждение суммы перед оплатой`
- `Скрин 4 — Инвойс Telegram Stars на оплату`
- `Скрин 5 — Сообщение об успешной оплате`

## 3) Сценарий, по которому делать скриншоты

1. Открой личку с `@break_remind_bot`, отправь `/thanks`.
2. Сделай скрин экрана с блоком выбора сумм.
3. Нажми на сумму (например, 50) и сделай скрин подтверждения.
4. Открой инвойс оплаты и сделай скрин страницы оплаты.
5. Заверши тестовый/боевой платёж и сделай скрин сообщения «Спасибо».

На каждом скрине должны быть видны название бота и цепочка «выбор → подтверждение → оплата → успех».

## 4) Что приложить дополнительно (если спросят)

- Ссылка на бота: `https://t.me/break_remind_bot`
- Краткое описание назначения платежа: `Добровольная поддержка разработки Telegram-бота Reminder`
- Категория: `Цифровой контент / онлайн-сервис`

## 5) Проверка перед отправкой

- [ ] `/ping` → актуальная версия бота
- [ ] `STARS_TIPS_ENABLED=true` (см. [ops-checklist.md](ops-checklist.md))
- [ ] Бот отвечает на `/start`
- [ ] Открывается `/thanks` и виден путь до подтверждения
- [ ] Минимум 3–5 скриншотов с последовательным сценарием
- [ ] На скриншотах видны `/about` или `/author` (автор и open source)
- [ ] На скриншотах нет чужих персональных данных
