# Напоминание на участника группы

Назначьте, **кому** в группе придёт упоминание при срабатывании.

## Способы

| # | Как | Пример |
|---|-----|--------|
| 1 | `/remind` + @ из списка | `/remind@бот @ivan завтра в 14:00 созвон` |
| 2 | @бот + @user + фраза | `@бот @ivan через час задача` |
| 3 | **Ответ** на сообщение + `/remind` | ответ на Ивана → `/remind@бот завтра задача` |

Приоритет: явный `@user` в тексте важнее, чем reply.

## Важно

- **@ из списка Telegram** (тап по имени) — не набор с клавиатуры.
- Участник должен **быть в группе** — иначе напоминание уйдёт создателю без тега.
- Подтверждение — в **личке** с ботом (кнопки «Создать / Отмена»).

## При срабатывании

В группе: `⏰ @ivan, **текст задачи**` — с кликабельным упоминанием.

## Голос

Можно ответить на сообщение человека **голосом** с фразой «завтра в 14:00 созвон» — бот возьмёт адресата из reply.

## Редактирование

`/edit 24 @petr завтра в 10:00 новый текст` — сменить и время, и адресата (если указан @).
