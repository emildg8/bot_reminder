# План: группы и каналы

## Принципы (best practices Telegram)

1. **Privacy Mode** — в группах надёжный путь: `/remind@бот`; @ только из меню Telegram.
2. **Разделение чатов** — личка / группа / канал: разные тексты, команды, UX.
3. **Collective delivery** — срабатывание в чат, управление (кнопки) в личке создателя.
4. **Права** — pause/clear/timezone в collective только у админов чата.
5. **Каналы** — только команды и явные mention; посты «от имени канала» без `/remind` не парсятся.
6. **Меню команд** — короткий scope для групп; отдельный для каналов при добавлении бота.

## Матрица поведения

| Действие | Личка | Группа | Канал |
|----------|-------|--------|-------|
| Создать текстом | фраза | @бот или `/remind` | `/remind` |
| Голос | да | @бот в caption* | нет (только команды) |
| Срабатывание | в чат + кнопки | в чат + личка | в канал + личка |
| Reply-клавиатура | да | нет | нет |
| pause/clear | всегда | админ | админ |

*если Telegram доставил update
