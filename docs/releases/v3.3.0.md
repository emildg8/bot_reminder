# v3.3.0 — Голос: «через два часа»

### Новое
- **«через два/три часа»** и **«каждые два часа»** — словами (STT)
- **Аудиофайлы** — mp3 и другие форматы как голос
- **Groq prompt** — лучше распознаёт русские напоминания
- **STT cleanup v3** — «запомни», «на помни», «ё»→«е»
- Короткий текст → подсказка перезаписать

### Примеры голосом
- «Через два часа созвон»
- «Завтра в два часа дня посмотреть фильм»
- «Каждые два часа встать»
