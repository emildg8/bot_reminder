# v3.11.0

## Каналы (фаза 3)

### Discussion group → канал
- При добавлении бота в канал сохраняется `linked_chat_id`
- `/remind` из группы обсуждений создаёт напоминание **в канале**
- В confirm: «Публикация — в канале (из группы обсуждений)»

### Подпись канала
- Срабатывание: `📢 Название канала` + текст напоминания

## Группы (фаза 4)

### @user с проверкой
- `get_chat_member` — упоминание только если user в чате
- Иначе: «не в этом чате» → напомню создателю

### `/list` в collective
- Без кнопок ✏️/🗑 на каждую строку
- Только пагинация и вкладки
- Подсказка: `/edit N`, `/pause`, управление в личке
