# v3.7.0

## Группы

При срабатывании напоминания: если сообщение в группу дошло, а личка с кнопками — нет, бот пишет в группу подсказку создателю (@username или ссылка).

## Дубликаты

«Созвон завтра» и «созвон послезавтра» больше не считаются дубликатом — сравнивается и расписание, не только текст.

## Редактирование

Фраза с несколькими weekly-временами при `/edit` работает как при создании: одно напоминание заменяется на несколько.

## NLP

- `через месяц` / `через 2 месяца` — относительное напоминание
- `15 июня в 10:00` — именная дата
- LLM-путь для «через N …» сохраняет `delay_seconds` (время при подтверждении)
