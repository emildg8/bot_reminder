# v3.2.0 — Умное распознавание голоса

### Голос и кружочки — готовый продукт

**STT cleanup**
- «14 00» → «14:00», «14-00» → «14:00»
- Убирает «э», «ну», «напомни мне» из начала фразы

**Парсинг времени (rule + LLM)**
- Слова: один…двадцать, «два часа дня» → 14:00
- «четырнадцать ноль ноль» → 14:00
- «завтра утром» → 9:00, «днём» → 14:00, «вечером» → 20:00
- «полтора дня» → 13:30

**UX**
- Подсказки при ошибке парсинга голоса
- Онбординг и /help с примерами голосовых фраз

**Рекомендация:** задай `GROQ_API_KEY` для лучшего STT на сервере.
