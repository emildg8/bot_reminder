# v3.8.0

## `/health` — авто-ремонт

При просроченных напоминаниях или рассинхроне с планировщиком (drift > 2) команда автоматически:
- перепланирует просроченные (с jitter, как при старте);
- добавит задачи для активных напоминаний без job в APScheduler.

В ответе: «Перепланировано: N просроч., M без задачи».

## Рефакторинг

- `bot/services/admin_access.py` — проверка админа
- `bot/services/deploy_info.py` — строка deploy sha
- `bot/services/health_status.py` — сбор метрик и формат ответа
- `bot/services/reminder_create.py` — создание + schedule из parsed items
- `repair_reminder_jobs()` в scheduler
