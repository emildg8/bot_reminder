# v3.5.1 — группы и расписание на несколько времён

### Исправлено
- **@бот в группе** — упоминание бота не путается с @user
- **«во вторник, среду… в 10:55, 16:10 и 21:00 Экспа»** — создаёт 3 напоминания с одной задачей

### Пример
```
@break_remind_bot Во вторник, среду, пятницу и субботу в 10.55, 16.10 и в 21.00 Экспа
```
