# v3.13.0

## Единый «ops chat» для collective

Discussion group и канал больше не расходятся в настройках.

### Операции из группы обсуждений → канал
- `/pause`, `/resume`, `/clear`
- `/export`, `/import`
- `/timezone` — часовой пояс канала

### UX
- «Применено к связанному каналу» в pause/clear/resume
- Onboarding TZ для discussion group — проверяет TZ канала
- `/status` одинаковый из команды и inline-меню

## Технически
- `collective_ops_chat_id()` в `timezone_ctx`
- `build_status_text()` в `chat_status.py`
