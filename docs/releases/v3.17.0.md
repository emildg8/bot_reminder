# v3.17.0

## Надёжность в группах (фаза 5B)

### Проблема
Кнопки меню и примеры иногда показывали Telegram «Ошибка»; падения хендлеров не объяснялись пользователю.

### Решение
- `callback.answer` сразу при нажатии — до edit/DB/DM
- Глобальный обработчик ошибок с alert «Попробуй ещё раз»
- `/sysinfo` показывает Group Privacy
- Fallback confirm в группе, если личка недоступна
