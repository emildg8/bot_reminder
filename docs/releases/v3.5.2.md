# v3.5.2 — группы: пиши через @бот

### Изменено
- При добавлении в группу бот показывает **реальный @username** и правило: без `@бот` он не видит сообщения
- `/help` и README обновлены под групповой режим

### Пример в группе
```
@break_remind_bot Во вторник, среду, пятницу и субботу в 10.55, 16.10 и в 21.00 Экспа
```
