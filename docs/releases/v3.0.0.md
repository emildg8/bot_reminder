# Напоминалка v3.0.0 — готовый продукт

Telegram-бот для напоминаний: текст, голос, кружочек. Личные чаты и группы.

## Основное

- **Напоминания** — разовые, интервальные, ежедневные, по дням недели
- **Парсинг фраз** — «через 3-4 часа», «через пару часов», «завтра в 14:00»
- **Дневник** — хронология событий за день
- **История** — срабатывания, отложения, «готово» (не удаляется за день)
- **Статистика** — выполнено, срабатывания, отложения за месяц
- **Отложить** — счётчик − / + и быстрые варианты (`/settings`)

## Меню

```
➕ Создать    📋 Список
📔 Дневник    📊 Статистика
⋯ Ещё
```

## Команды

`/start` · `/list` · `/history` · `/journal` · `/stats` · `/settings` · `/about` · `/help`

## Деплой

Startup на Wispbyte: `bash start.sh`  
Подробнее: [.github/DEPLOY.md](https://github.com/emildg8/bot_reminder/blob/main/.github/DEPLOY.md)

## Требования

- Python 3.11+
- `BOT_TOKEN` от [@BotFather](https://t.me/BotFather)
- ffmpeg — для голосовых и кружочков
