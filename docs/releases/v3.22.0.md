# v3.22.0 — Уточнение «завтра в 2»

## Изменения
- **«завтра в 2»** — только две кнопки: ☀️ 14:00 (дня) и 🌙 02:00 (ночи)
- **Редактирование** (`/edit` или режим edit) — те же уточнения, что при создании
- Legacy-кнопка 🌅 09:00 на старых сообщениях — alert «Выбери день или ночь»

## Без изменений
- **«завтра созвон»** — по-прежнему 🌅 09:00 / ☀️ 14:00 / 🌆 18:00
