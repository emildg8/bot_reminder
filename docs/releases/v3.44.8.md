# v3.44.8 — groups: assignee auto-pick + прозрачность

**Дата:** 2026-06-02

## Группы / assignee

- Точный детект `@бот` (без ложных `@mybot123`).
- **`pick="auto"`** (default): один `@user` — первый; несколько — ближайший к времени.
- Форматы: `@бот@user`, `@бот + @user`, `для/кому/напомни @user`, текст до `@бот` в середине фразы.
- Время: `в 18.20`, дни недели, `в 1430`, правка сообщения с `@бот` (edited_message).
- **Confirm в личке:** строка «Кому» + пояснение, если @user несколько.
- **В группе до confirm:** `👌 👤 @user · через 1 мин · тест` → подтверди в личке.
- Подсказка при ошибке парса в группе (форматы `@бот`, `/remind`, время).

## Clean текста

- Пунктуация после удаления @; пустые `()`, `[]`, `{}`.
