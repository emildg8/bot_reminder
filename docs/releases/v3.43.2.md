# v3.43.2 — Stars tips расширение

**Дата:** 2026-06-03

## Что нового

- **✨ Другая сумма** — ввод своей суммы Stars (min/max в env)
- Presets по умолчанию: 25 / 50 / 100 / 250 / 500
- `/thanks` показывает историю поддержки
- Мягкий nudge после «✅ Готово» (cooldown, только без донатов)
- `/status` — строка «Поддержать автора: /thanks»

## Env

```env
STARS_TIP_PRESETS=25,50,100,250,500
STARS_TIP_MIN=1
STARS_TIP_MAX=2500
STARS_TIP_NUDGE_ENABLED=true
STARS_TIP_NUDGE_DAYS=14
```
