# v3.40.0 — Admin / user mode

**Дата:** 2026-06-02 · **Тесты:** 401

## Режим admin / user

- `ADMIN_TELEGRAM_IDS` + встроенный admin — переключение «как админ» / «как пользователь»
- БД: `users.admin_tools_enabled` (миграция `20260601_0002`)
- User-mode: лимиты Free, без `/health`, `/sysinfo`, ops-команд; умный отказ
- `/adminmode`, reply-кнопки, меню команд per-admin

## Панель `/admin`

| Команда | Назначение |
|---------|------------|
| `/userinfo` | Карточка (ID или reply), Pro, напоминания |
| `/userfind` | Поиск ID по фрагменту |
| `/broadcast` | Превью, фильтры, «Себе», до 100 |
| `/grantpro` / `/revokepro` | ID или reply |
| `/adminstats` | Расширенная статистика |
| `/adminlog` | Журнал действий (in-memory) |
| `/admins` | Список админов |
