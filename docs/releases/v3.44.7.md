## Парсинг assignee

- `extract_username_anywhere(..., pick="first" | "last")` — выбор первого/последнего `@user` при нескольких упоминаниях.
- `extract_username_candidates()` — возврат всех кандидатов (`@user`) + clean без mention для будущей A/B-логики.

## UX clean-текста

- После удаления mention больше нет артефактов вида `тест !`; теперь нормализуется до `тест!`.
- Поддержаны хвостовые знаки после mention: `@user!`, `@user?`, `@user,`.
