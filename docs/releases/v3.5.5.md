# v3.5.5

## Проблема

В группах `/ping@бот` работал, а текст `@break_remind_bot …` — нет. При Privacy Mode Telegram доставляет боту только **команды** и сообщения с **mention entity** (когда @ выбран из меню). Ручной ввод `@username` часто не создаёт entity — бот просто не получает update.

## Решение

- Команда **`/remind@бот <фраза>`** — всегда доходит до бота
- Обновлены приветствие в группе, `/help` и меню команд
- Текстовый handler в группах обрабатывает только сообщения с упоминанием бота

## Пример

```
/remind@break_remind_bot Во вторник, среду, пятницу и субботу в 10.55, 16.10 и в 21.00 Экспа
```
