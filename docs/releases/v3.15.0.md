# v3.15.0

## Меньше шума в групповом режиме

### Было (на каждое действие)
- Confirm: длинная подсказка в группе + заголовок в личке + «Подтверди действие»
- Создано: «Готово» + суффикс «кнопки в личке» + **ещё одно** сообщение с той же подсказкой
- Срабатывание: текст в группе + «Управление в личке» + полный дубликат в личке

### Стало
- **Группа при confirm:** `👌 Подтвердите в личке с ботом`
- **Личка при confirm:** `📣 Название группы` + расписание + кнопки
- **Группа при создании:** `✅ @user · #21 · когда · текст` (одна строка)
- **Личка при создании:** `✅ Готово! #21` без повторов
- **Срабатывание в личке:** `⏰ #21 · текст` + кнопки
- **Срабатывание в группе:** только текст напоминания

Подсказка про /start — только если личка недоступна.
