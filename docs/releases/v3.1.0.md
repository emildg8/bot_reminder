# v3.1.0 — Голос и кружочки

Telegram-бот для напоминаний: текст, **голос**, **кружочек**. Личные чаты и группы.

## Что нового

### 🎤 Голосовые сообщения
Отправь голос одной фразой — бот распознает речь и предложит создать напоминание.

### 🔵 Кружочки (video note)
То же самое через кружочек — удобно, когда не хочется печатать.

### Распознавание речи (STT)
1. **Groq Whisper** — быстро и без нагрузки на сервер (нужен `GROQ_API_KEY`)
2. **Локальный Whisper** — fallback, модель `tiny`
3. **Yandex SpeechKit** — опционально (`USE_YANDEX_STT=true`)

### Сервер
- `ffmpeg` нужен для локального Whisper и как резерв для кружочков
- `start.sh` автоматически пытается установить ffmpeg
- Docker-образ уже включает ffmpeg

## Примеры фраз

- Голосом: «Напомни через час выпить таблетки»
- Кружочком: «Каждый день в девять зарядка»

## Команды

`/start` `/list` `/history` `/journal` `/stats` `/settings` `/about` `/help`

## Требования

- Python 3.11+
- `GROQ_API_KEY` рекомендуется для голоса и кружочков
- ffmpeg — для локального fallback
