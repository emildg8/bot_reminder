# v3.6.0

## UX в группах

- При подтверждении: «Напоминание в группе · кнопки управления — в личке»
- После создания: «Сработает в группе · кнопки — в личке» + подсказка написать бота в личку
- Reply-клавиатура меню не показывается в группах (только в личке)

## Парсинг и карточки

- «Через N минут» отображается как «через N мин от подтверждения» (в confirm и после создания)
- Batch weekly: заголовок «N еженедельных напоминания (по одному на время)»

## Ops

- `/health`: просроченные напоминания с учётом timezone; deploy sha local → GitHub
- `/restart` и автообновление: корректная остановка polling перед execv
